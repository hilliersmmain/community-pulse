# Screenshots

This directory contains screenshots of the Community Pulse dashboard.

## Required Screenshots

1. **dashboard_main.png** - Main dashboard view showing KPI metrics
2. **cleaning_pipeline.png** - Data cleaning execution log
3. **analytics_charts.png** - Interactive visualizations
4. **before_after.png** - Data quality comparison

## How to Capture

1. Run the application locally: `streamlit run app.py`
2. Generate sample data
3. Run the cleaning pipeline
4. Navigate through all tabs
5. Take screenshots using your OS screenshot tool
6. Save images with the filenames listed above

## Image Requirements

- Format: PNG
- Resolution: At least 1920x1080
- Clear, professional appearance
- Include relevant UI elements
